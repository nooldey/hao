## 一个基于parcel打包工具的vue 前端导航站点

[![Build Status](https://travis-ci.org/zeroNav/hao.svg?branch=master)](https://travis-ci.org/zeroNav/hao)

**How To Use**

Clone this git, and run script like this:

```bash
    npm install
    npm run dev
```

For Local Dest:
    `npm run build`

For production:
    `npm run prod`

**Happy Using Vuejs**
